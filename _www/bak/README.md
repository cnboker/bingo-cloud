RDS(Raspberry Digital Signage)是一款专门为Raspberry Pi 4设计的一款系统，利用Raspberry Pi 4作为基板实现数字标牌的常见应用，RDS终端app基于[WebOS OSE](https://www.webosose.org/)操作系统开发，实现图片，视频，时钟，天气，网页，pdf等播放功能.
